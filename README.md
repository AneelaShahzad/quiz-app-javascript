# 🧠 Quiz App (JavaScript)

A simple interactive quiz application built using **HTML, CSS, and vanilla JavaScript**.  
This project was created as part of my frontend learning journey to practice core JavaScript concepts during my second semester of comp sci.

---

## 🚀 Features

- 👤 User enters name before starting the quiz
- ❓ 5 multiple-choice questions
- ⏱️ 10-second timer for each question
- 🔄 Automatic transition to next question
- 🎯 Score display at the end with user name
- 🎉 Special message for full score (5/5)
- 🔁 Option to restart the quiz
- 🔊 Sound effects for better user experience

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 📸 Project Flow

1. User enters their name  
2. Clicks "Start Quiz"  
3. Questions appear one by one  
4. Each question has 10 seconds to answer  
5. Auto move to next question (with or without answer)  
6. Final score is displayed  
7. Option to retry the quiz
